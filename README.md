# World Happiness Report Dashboard

An interactive Streamlit dashboard for visualizing and analyzing global happiness data from the World Happiness Report (2005-2023).

## Features

- **Interactive Filters**: Filter by year, region, and happiness score range
- **Comprehensive Visualizations**:
  - Univariate and bivariate analysis
  - Interactive bubble charts with animation
  - Choropleth world maps
  - Time series trends
  - Top countries ranking
- **Multi-tab Interface**: Organized analysis by category
- **Responsive Design**: Works on desktop and mobile
- **Data Export**: Download filtered data as CSV
